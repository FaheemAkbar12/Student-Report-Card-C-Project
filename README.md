# Student Report Card Project

A simple C++ console application that collects student details and marks, then prints a formatted report card with percentage, grade, and remark.

## Files

- `2 new.cpp` - main program source
- `2 new.exe` - compiled Windows binary
- `Student_Report_Card_Project_Report_F.pdf` - project report document

## Build

Use any C++ compiler, for example with g++:

```bash
g++ "2 new.cpp" -o student_report_card
```

## Run

```bash
./student_report_card
```

On Windows, run the generated `.exe` file directly.
